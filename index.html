CycleGans for Underwater Plastic Detection and cleaning

An efficient and scalable architecture for underwater plastic detection and cleaning using Underwater Autonomous Vehicle (AUV) and CycleGans as Data Augmentation technique to convert in air plastic to underwater style. Project is in progress. I have attached videos of training cyclegans on my own dataset for converting in-air plastic to underwater style. As well as images for training around 60 epochs with descending G-loss and D-loss curves per epoch. Can be found here - Project Link

Underwater Plastic Detection & Cleaning using Computer Vision & IOT

An AUV for underwater garbage detection, collection and cleaning using Computer Vision and IOT. It uses tflite model for garbage detection with almost 80% accuracy. The model is trained on 1500 images of 5 classes namely straws, bottle, cans, bags and cups. The designed system would be able to detect the plastic, calculate distance using triangulation and OpenCV and then collect the detected garbage using Robotic Arm. Collected garbage would be compressed and collected in attached net. Can be found here - Project Link

Multiclass Image classication on UC-Merced LandUse Dataset

This project aims at classification of remote sensing image dataset. The model developed for classification is a fusion model of spatial features with dct features.3-layer fusion model of cnn is used with dct and lbp to improve the accuracy of prediction. Various accuracy enchancing techniques like augmentation and feature level fusion of dct and lbp are used Project contains various ipynb notebooks tried on the dataset. Project can be found here - Project Link

IOT based Smart Home using Face Recognition

This project provides controlling and monitoring of home appliances as well as provides security from unknown persons. We proposed a system for Smart Home Automation technique. To design this system, we used a Raspberry Pi module and Computer Vision techniques, OpenCV and image processing algorithms for Face Recognition. Project Link
